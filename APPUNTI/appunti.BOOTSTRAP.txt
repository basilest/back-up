    _____________________________ WHAT IT IS & HISTORY
             Bootstrap is
                 standared framework for
                       - FRONT-END HTML/ CSS/ JAVASCRIPT
                       - aim to self adjust to different devices (eg. MOBILE (small phones) upto large desktops)
                       - uses jquery for cross browser compatibility

                 Deeveloped at Twitter then released as open source on 2011

    _____________________________ WHERE FIND IT

      - Download Bootstrap from getbootstrap.com
      - Include Bootstrap from a CDN  (like jquery)


      It mainly consists of 3 things to add in the HTML header
          - 1 css file
          - 1 js file
          - jquery

          ex:
            <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
            <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>
            <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>
